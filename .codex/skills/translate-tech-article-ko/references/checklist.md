# Translation Checklist

최종 응답 또는 파일 저장 전에 아래를 확인한다.

- 원문 URL이 확인되었는가
- frontmatter 바로 아래 첫 bullet에 원문 링크 안내가 들어갔는가
- `- 원문은 [출처명의 원 글](URL)을 확인해주세요.` 형식을 지켰는가
- `title`이 영문 원제인지 확인했는가
- `date`의 기준이 명확한가
- 문체가 설명형 기술 블로그 톤으로 유지되는가
- 핵심 용어만 선별적으로 영어 병기를 했는가
- 직역투 문장이 남아 있지 않은가
- 원문에 없는 주장이나 해설을 추가하지 않았는가
- 헤딩 언어 선택이 문서 전체에서 일관적인가
- 링크, 이미지, 목록, 코드 블록이 깨지지 않았는가
- 게시 가능한 Markdown으로 바로 사용할 수 있는가
